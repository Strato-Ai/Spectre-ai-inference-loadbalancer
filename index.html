How it works

Request to Response

Every inference request flows through a five-stage pipeline — authenticated, scored, routed, and served.

01
Ingress
Client sends an HTTPS request to NGINX on port 443. SSL terminates here. The only public entry point into the cluster.
02
Auth
NGINX issues an auth_request to the FastAPI Auth Service. SHA-256 hashed API keys. Invalid requests rejected before touching a backend.
03
Score
The Routing Sidecar polls each backend's GPU monitor agent and computes a weighted load score in real time.
04
Route
NGINX forwards to the lowest-scoring backend. Model-family routing via URL path prefix — no JSON body inspection required.
05
Serve
The winning multiple compute backend handles inference. Response streams back through NGINX to the client.
Features

Built for Real GPU Infrastructure

Spectre handles heterogeneous hardware, model-aware routing, and secure multi-client access without complex orchestration overhead.

⚖️
GPU-Aware Load Scoring
Weighted composite score: GPU utilization (50%), VRAM usage (30%), active requests (20%). Lowest score wins the next request.
Real-time
🧠
Model-Family Routing
Route by URL path prefix or X-Model-Hint header. No JSON body inspection — fast and simple.
Path-based
🔑
API Key Authentication
Per-request validation via NGINX auth_request. SHA-256 hashed keys. Zero trust — every request authenticated.
Secure
🖥️
Heterogeneous Hardware
Native support for NVIDIA GPUs via nvidia-smi and Apple Silicon via macmon — sudoless, same scoring formula.
NVIDIA + Apple Silicon
🏠
Homelab to Enterprise
Runs on a single-node homelab or deploys to AWS via included Terraform modules. Private subnet isolation for GPU backends.
Terraform included
📋
10-Point Test Suite
Automated validation covering auth, routing, health polling, load scoring, failover, and end-to-end inference. Know it works before you ship.
Automated

Quick Start

Up in Minutes

Clone, configure your backend IPs, and run the bootstrap script. The test suite will confirm everything is working.

Clone & bootstrap
# Clone the repo git clone https://github.com/strato-ai/spectre.git cd spectre # Edit backend IPs in CLAUDE.md or config/backends.conf # then run the load balancer bootstrap bash scripts/bootstrap-loadbalancer.sh # On each GPU backend bash scripts/bootstrap-backend.sh # Validate the full stack bash scripts/test-suite.sh
Send an inference request
curl -X POST https://10.0.1.5/v1/chat/completions \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"model":"qwen3-4b","messages":[{"role":"user","content":"Hello"}]}'
